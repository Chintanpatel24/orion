# Roadmap

## 0.2

- Native desktop shell
- Refined Orion logo and icon assets
- Persistent project folder memory
- File explorer
- Multiple tabs
- Lightweight syntax highlighting
- Agent Git Review
- GitHub-style side-by-side diff view
- Done and Not done review markers
- Stage, unstage, and commit actions
- Command palette
- Settings
- Install scripts
- GitHub CI

## 0.3

- Better diff alignment for modified lines
- File create, rename, and delete in Project panel
- Recent projects list
- Reviewed-file grouping
- Commit history viewer
- Branch switcher
- Improved syntax highlighting
- Configurable keybindings

## 0.4

- Optional language server integration
- Optional agent task interface
- Project task definitions that open in external terminal or CI instead of an internal output panel
- Integrated diagnostics panel

## Long term

- Permissioned agent API
- Sandboxed task execution helpers
- Signed releases
- Native package formats for Windows, Linux, and macOS
