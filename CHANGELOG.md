# Changelog

## 0.2.0

- Renamed the project to Orion IDE
- Added Orion logo, icon, and interface preview assets
- Added low-power mode and highlighting limits for very old hardware
- Tuned syntax highlighting to avoid per-frame keyword-set allocation
- Rebuilt Orion as a native Rust desktop IDE
- Added workspace explorer
- Added multi-tab editor
- Added syntax highlighting for low-level languages
- Removed the code-runner output panel from the main workflow
- Added built-in GUI Git Review with side-by-side diffs
- Added Done and Not done review markers
- Added stage, unstage, and commit actions
- Added command palette
- Added settings window
- Added GitHub-ready repository files
- Added install scripts for Windows, Linux, and macOS
